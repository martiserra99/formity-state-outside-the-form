# Formity — State Outside the Form

Starting point for the [State outside the form](https://formity.app/docs/guides/state-outside-the-form) guide on the Formity docs.

## Overview

This repository contains a multi-step checkout form built with Formity and React Hook Form. It collects contact details, a shipping address, a delivery method, and payment information. The form state is not yet accessible outside the form. The guide walks you through exposing it so you can use it anywhere — for example, to show a live summary panel alongside the form.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-state-outside-the-form
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
